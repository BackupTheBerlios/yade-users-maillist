xlC_r is probably plain c compiler, whereas you would theoretically need 
c++ compiler. Besides that, and more importantly, yade will not compile 
anything else than g++; you will probably have to install that one as 
well. You can do it from sources, but it is involved. Better if you ask 
the admin to install that for you.

http://www.perzl.org/aix/index.php?n=Main.Gcc can help you. Another 
problem can be binary compatibility of c++ libs between gcc and the AIX 
compiler, but you will see if that works or not.

Sorry for bad news, get back if you need more (help, not bad news ;-) )
