first things first. Please make sure the we don't have any version
conflicts here.

Then please delete your ~/YADE/ (backup anything you might need later)

versions, than are incompatibile, perhaps we would need some kind of
binary lib* versioning to prevent that. It became my habit to delete
config files and old binary files in case of some inexplicable
problems.
